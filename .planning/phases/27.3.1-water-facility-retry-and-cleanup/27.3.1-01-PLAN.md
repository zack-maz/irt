    1. Confirm Overpass is responsive:
       ```bash
       curl -sS --max-time 10 -X POST https://overpass-api.de/api/interpreter \
         -d 'data=[out:json][timeout:5];node(51.5074,-0.1278,51.5075,-0.1277);out;' \
         | head -c 200
       ```
       Expect JSON. If you get HTML, timeout, or `runtime error: Query timed out`, STOP — Overpass is still down. Do not proceed. Re-run Task 1 later.

    2. Clear the dev file cache:
       ```bash
       rm -f .dev-cache/water-facilities.json
       ```

    3. Clear Redis keys (via Upstash REST API — use the credentials in `.env.local`):
       ```bash
       # Source env vars first
       set -a; source .env.local; set +a

       curl -sS -X POST "$UPSTASH_REDIS_REST_URL/del/water:facilities" \
         -H "Authorization: Bearer $UPSTASH_REDIS_REST_TOKEN"
       curl -sS -X POST "$UPSTASH_REDIS_REST_URL/del/water:precip" \
         -H "Authorization: Bearer $UPSTASH_REDIS_REST_TOKEN"
       ```
       Expect `{"result":1}` or `{"result":0}` (0 = key did not exist, still a success).

    4. Start the dev server (or confirm it is already running):
       ```bash
       npm run dev:server
       ```
       Wait for the `server listening on :3001` line in the output.

    5. Hit the refresh endpoint and save the response (may take 60-90s for three Overpass queries to complete):
       ```bash
       curl -sS --max-time 180 "http://localhost:3001/api/water?refresh=true" \
         -o /tmp/water-refresh.json
       echo "HTTP bytes: $(wc -c < /tmp/water-refresh.json)"
       cat /tmp/water-refresh.json | python3 -m json.tool | head -40
       ```

    6. Extract per-type counts and rejection buckets:
       ```bash
       cat /tmp/water-refresh.json | python3 -c "
       import json, sys
       r = json.load(sys.stdin)
       total = len(r['data'])
       types = {}
       for f in r['data']:
           types[f['facilityType']] = types.get(f['facilityType'], 0) + 1
       print('Total facilities:', total)
       print('By type (on-map):', types)
       print()
       fs = r.get('filterStats')
       if fs:
           print('rawCounts:', fs['rawCounts'])
           print('filteredCounts:', fs['filteredCounts'])
           print('rejections:', fs['rejections'])
           print('enrichment:', fs['enrichment'])
           print('scoreHistogram:', fs['scoreHistogram'])
       else:
           print('NO filterStats — response was cached. STOP and re-check step 3 (cache clear).')
       "
       ```

    7. Paste all output from steps 5 and 6 into the REFRESH-LOG.md template (Task 2 creates the template).
