**1. Create `src/components/detail/BreadcrumbRow.tsx`:**

A thin, horizontally scrollable row showing previous stack entries. Per user decisions:
- Appears only when stack has entries (slides in/out vertically)
- Shows previous entries only (current view is NOT in the breadcrumb)
- Labels: truncated with ellipsis for long names
- Uniform muted gray text, no entity color dots, `>` chevron separators
- Left arrow button goes back one step
- Last (most recent) segment is also clickable (same as back)
- Deeper segments are display-only (no click handler)
- Horizontal scroll when breadcrumb exceeds panel width (overflow-x-auto + whitespace-nowrap)

Props:
```typescript
interface BreadcrumbRowProps {
  stack: PanelView[];
  onBack: () => void;
}
```

The `onBack` handler is used for both the arrow button and clicking the last segment. An arrow-left icon (use a simple SVG inline or unicode arrow `\u2190`). Row height: ~28px (py-1.5). Typography: text-[10px]. Max label width per segment: max-w-[120px] with truncate. Use `animate-breadcrumb-enter` class from app.css.

d) Add `goBack` handler that also flies the map to the restored view:
```typescript
const handleGoBack = useCallback(() => {
  // Read the entry we're about to restore BEFORE calling goBack
  const stack = useUIStore.getState().navigationStack;
  const prev = stack[stack.length - 1];
  goBack();
  // Fly to the restored view's coordinates
  if (prev) {
    if (prev.cluster) {
      setFlyToTarget({ lng: prev.cluster.centroidLng, lat: prev.cluster.centroidLat, zoom: 8 });
    } else if (prev.entityId) {
      // Look up entity coordinates from stores
      const entity = findEntityById(prev.entityId); // use the non-reactive lookup from panelLabel
      if (entity) {
        setFlyToTarget({ lng: entity.lng, lat: entity.lat, zoom: 10 });
      }
    }
  }
}, [goBack, setFlyToTarget]);
```

Import `useNotificationStore` for `setFlyToTarget` (already used in some components; add selector if not present). Import `findEntityById` from `@/lib/panelLabel` if exported, or inline the store lookup.
