## Decisions Made

- Exported `FACILITY_TYPE_LABELS` from `overpass-water.ts` for shared use -- avoids duplicating the labels map
- `labelUnnamedFacilities` runs between fetch and cache write so Redis stores enriched labels (geocoding only runs on 24h cold cache)
- 500 unnamed facility cap per cold cache run with 1050ms delay between Nominatim calls
- Stress thresholds use simple thirds of compositeHealth: <=0.33 High, <=0.66 Medium, >0.66 Low
- Attacked/healthy filter reuses existing `destroyedIds` Set (5km proximity to destructive events)
