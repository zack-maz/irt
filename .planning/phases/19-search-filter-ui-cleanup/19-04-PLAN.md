**1. Create `src/hooks/useDraggable.ts`:**
Reusable drag hook using Pointer Events API.
```typescript
interface DraggableOptions {
  storageKey: string;          // localStorage key for position persistence
  defaultPosition: { x: number; y: number }; // default position (top-right for markets)
  bounds?: { minX: number; minY: number; maxX: number; maxY: number }; // viewport constraints
}

interface DraggableReturn {
  position: { x: number; y: number };
  isDragging: boolean;
  handleProps: {
    onPointerDown: (e: React.PointerEvent) => void;
    onPointerMove: (e: React.PointerEvent) => void;
    onPointerUp: (e: React.PointerEvent) => void;
    style: { touchAction: 'none'; cursor: 'grab' | 'grabbing' };
  };
  resetPosition: () => void;
}
```

Implementation:
- Load initial position from localStorage (fallback to defaultPosition)
- onPointerDown: capture pointer via `setPointerCapture`, record drag start offset
- onPointerMove: compute new position, constrain to viewport bounds (clamp to 0..window.innerWidth-panelWidth, topbar height..window.innerHeight-100)
- onPointerUp: release pointer capture, persist position to localStorage
- resetPosition: set position to defaultPosition, remove from localStorage
- Use `useCallback` for all handlers to prevent re-renders
