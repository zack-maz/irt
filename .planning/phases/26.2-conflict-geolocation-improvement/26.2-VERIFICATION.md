# Phase 26.2: Conflict Geolocation Improvement Verification Report

**Phase Goal:** Reduce misplaced conflict events on the map by adding NLP-based cross-validation to the GDELT pipeline. Extract actors and locations from article titles using compromise (free, local, already installed), cross-check against GDELT's geocoded coordinates, reject mismatches, and relocate centroid events to specific cities when NLP identifies a place name. No paid API fees — all NLP runs locally in the serverless function.
