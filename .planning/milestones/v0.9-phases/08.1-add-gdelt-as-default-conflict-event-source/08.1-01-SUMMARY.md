## Decisions Made
- Used adm-zip for ZIP decompression because Node.js built-in zlib handles gzip/deflate but cannot decompress ZIP archives (different container format)
- HTTP URLs for GDELT data server due to verified TLS certificate issues (ERR_TLS_CERT_ALTNAME_INVALID)
- FIPS 10-4 country codes (16 Greater Middle East countries) matching GDELT's ActionGeo_CountryCode format
- Tab delimiter for parsing despite .CSV file extension (GDELT uses tab-delimited format)
- CAMEO root code classification: 18 (Assault) -> drone, 19 (Fight) -> missile, 20 (Mass violence) -> missile
- gdelt- prefix on entity IDs to distinguish from ACLED events
- fatalities always 0 (GDELT does not track fatalities)
