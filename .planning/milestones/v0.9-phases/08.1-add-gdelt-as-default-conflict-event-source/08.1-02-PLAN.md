**server/config.ts:** Make ACLED credentials optional to prevent server startup crash. Change the `acled` config block from:

```typescript
acled: {
  email: required('ACLED_EMAIL'),
  password: required('ACLED_PASSWORD'),
}
```

to:

```typescript
acled: {
  email: process.env.ACLED_EMAIL ?? '',
  password: process.env.ACLED_PASSWORD ?? '',
}
```

This keeps the config shape intact (ACLED adapter code still compiles) but does not crash on startup when env vars are absent.

**server/**tests**/server.test.ts:** Add a mock for the GDELT adapter alongside the existing adapter mocks:

